# Miniguia de Estudos: Explorando o TDAH com IA 🧠

## 🎯 Contexto e Objetivos
Este projeto foi desenvolvido como parte de um desafio da DIO para explorar o uso da IA (NotebookLM) na organização de conhecimentos. O tema escolhido foi o **TDAH (Transtorno do Déficit de Atenção com Hiperatividade)**, com o objetivo de entender melhor os seus mecanismos e criar estratégias de estudo eficazes.
