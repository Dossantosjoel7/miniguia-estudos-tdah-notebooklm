# Miniguia de Estudos: Explorando o TDAH com IA 🧠

## 🎯 Contexto e Objetivos

Este projeto foi desenvolvido como parte de um desafio da DIO para explorar o uso da IA (NotebookLM) na organização de conhecimentos. O tema escolhido foi o **TDAH (Transtorno do Déficit de Atenção com Hiperatividade)**, escolhi este tema com interesse pessoal,porque tenho sintomas não-diagnosticados ligados ao cérebro neurodivergente desde criança, então venho com objetivo de entender melhor acerca do TDAH, como funciona nos adultos e seus possiveis diagnostico.
